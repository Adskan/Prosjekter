# Prosjekt-tictactoe

```
Et simpelt prosjekt jeg lagde i sommer 2022, når jeg skulle lære meg litt mer om javascript, html og css.
Når jeg begynte kunne ingenting om noen av de, men etter å ha laget dette føler jeg at jeg har en bedre forståelse for mye grunnleggende for de 3 språkene. Selv om jeg ikke kan alt om språkene enda, var det et svært givende prosjekt som ga meg mye motivasjon bak å lære flere programmeringsspråk, samt. kunnskap bak tilnærmingen til språk. 
```
